# 💰 Учёт расходов — BMW Finance

## Описание проекта

Хотите знать, куда уходят деньги? Наше приложение поможет вам учитывать доходы и расходы, планировать бюджет и достигать финансовых целей.

Простой интерфейс, удобная аналитика и полный контроль над личными финансами — всё в одном приложении.

### Основные возможности:
- Добавление доходов и расходов в один клик
- Наглядные графики и диаграммы
- Категории расходов (продукты, транспорт, развлечения и др.)
- Разделение на доходы и расходы
- Поиск и фильтрация транзакций
- Статистика по категориям

---

## Функционал

- ✅ Возможность вносить данные (сумма, дата, описание, категория, счёт)
- ✅ Просмотр истории расходов и доходов
- ✅ Статистика по категориям (дни, месяцы)
- ✅ Поиск транзакций по описанию
- ✅ Фильтрация по категории и дате
- ✅ Управление счетами (карта/наличные)
- ✅ Управление категориями (расход/доход)
- ✅ Авторизация и регистрация пользователей
- ✅ JWT-аутентификация

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Quiten-404/Progect.git
cd Progect
```

### 2. Создание виртуального окружения

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac/WSL:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Инициализация базы данных
```bash
python -c "from src.database import init_database; init_database()"
```

### 5. Запуск сервера
```bash
python -m uvicorn src.api_db:app --reload
```

### 6. Открыть в браузере
```
http://127.0.0.1:8000/web/register
```

## 🔌 API документация

После запуска сервера доступны:

- **Swagger UI**: http://127.0.0.1:8000/docs
- **ReDoc**: http://127.0.0.1:8000/redoc

### Основные API эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/register` | Регистрация |
| POST | `/login` | Вход (JWT) |
| GET | `/accounts/{user_id}` | Счета пользователя |
| POST | `/accounts` | Создать счёт |
| GET | `/transactions/{user_id}` | Транзакции пользователя |
| POST | `/transactions` | Создать транзакцию |
| GET | `/categories/{user_id}` | Категории пользователя |
| POST | `/categories` | Создать категорию |
