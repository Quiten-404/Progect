# Учет расходдов

# Описание проекта
Описание:Хотите знать, куда уходят деньги? Наше приложение поможет вам учитывать доходы и расходы, планировать бюджет и достигать финансовых целей.
Простой интерфейс, удобная аналитика и полный контроль над личными финансами — всё в одном приложении.
Основные возможности:
Добавление доходов и расходов в один клик. 
Наглядные графики и диаграммы.
Категории расходов (продукты, транспорт, развлечения и др.) 
Бюджет на месяц с уведомлениями.
# Функцианал

- Возмодность вносить данные (Сумма, Дата, Место, Описание)
- Просмотр истории Расходов
- Статистика по (дням, неделям)
- Рекомендации
